# JacWin10
JacWin10 XFCE Theme
Just a Copy Windows10
Created by Luiz F. Nunes

Theme XFWM4 to Window Manager and GTK2 Theme for XFCE

To install unzip the folder to directory of themes of system:

/usr/share/themes/
or
~/.themes/
