# JacWin10
JacWin10 XFCE Theme<br/>
Just a Copy Windows10<br/>
Created by Luiz F. Nunes<br/>

Theme XFWM4 to Window Manager and GTK2 Theme for XFCE<br/>

To install unzip the folder to directory of themes of system:

/usr/share/themes/
or
~/.themes/
